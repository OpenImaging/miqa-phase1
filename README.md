# miqa

## Project setup
```
npm install
```

## Prepare data
Find a .nifti file and name it image.nii.gz and place it under the `public` directory

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
